## Condition if else 
Оператор if-else используется для выполнения как истинной, 
так и ложной части данного условия. Если условие истинно,
 выполняется код блока if, а если условие ложно, 
 выполняется код блока else.
В JavaScript у нас есть следующие условные операторы: • Используйте if, чтобы указать блок кода,
 который будет выполнен, если указанное условие истинно.
  • Используйте else, чтобы указать блок кода, который будет выполнен,
   если то же условие является ложным. • Используйте else if,
    чтобы указать новое условие для проверки, если первое условие ложно.


## Ternary Operator
Альтернатива оператору if/else, тернарный оператор позволяет разработчикам JavaScript писать краткие условные операторы.
 Он записывается как «?:» и принимает три операнда; логическое условие, значение,
  возвращаемое в случае истинности, и значение, возвращаемое в случае ложности.



  ## Function declaration
  Оператор for создает цикл, состоящий из трех необязательных выражений, заключенных в круглые скобки и разделенных точкой с запятой, за которыми следует оператор (обычно оператор блока), который должен выполняться в цикле.


  ## Switch case 
  Оператор переключения оценивает выражение, сопоставляя значение выражения с рядом предложений case,
   и выполняет операторы после первого предложения case с совпадающим значением
   , пока не встретится оператор прерывания. Предложение по умолчанию оператора переключателя будет переключено, 
   если ни один регистр не соответствует значению выражения.

## Условные операторы

Условные операторы в JavaScript позволяют выполнять различные действия в зависимости от условий. Это основа принятия решений в программировании. В JavaScript есть несколько типов условных операторов:

- **`if`** — основной оператор для проверки условий.
- **`else if`** — используется для добавления дополнительных условий.
- **`else`** — выполняется, если не выполняется ни одно из условий.

### Пример кода:
```javascript
let age = 25;

if (age < 18) {
    console.log("Вы слишком молоды.");
} else if (age >= 18 && age < 60) {
    console.log("Добро пожаловать!");
} else {
    console.log("Пора отдыхать.");
}
